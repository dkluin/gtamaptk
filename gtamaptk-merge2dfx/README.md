# gtamaptk-merge2dfx
This tool reads 2dfx data from DFFs in two different folders and combines them before writing out new DFFs with said combined 2dfx data.

## Usage
You can use the the tool as follows:

```
gtamaptk-merge2dfx [path to folder containing 1st DFF files] [path to folder containing 2nd DFF files] [path to output DFF folder]
```
