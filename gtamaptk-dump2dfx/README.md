# gtamaptk-dump2dfx
This tool extracts the 2dfx data to a separate folder for a given set of DFF files, allowing you to view them with DK22Pac's tool.

## Usage
You can use the the tool as follows:

```
gtamaptk-dump2dfx [path to folder containing DFF files] [output folder where the *.2dfx files will be written to]
```
